# Чемпіонат світу 2026 — сітка плей-оф ⚽

Самодостатня HTML-сторінка з сіткою плей-оф ЧС-2026 (USA / Canada / Mexico):
сітка з кубком, розклад «Коли дивитись» (час за CET/CEST) і таблиці груп.
Оновлюється автоматично, без бекенду й без ключів.

## Як це працює

- Один файл [`index.html`](index.html) — HTML + CSS + JS, без залежностей і без збірки.
- Дані тягнуться з публічного JSON проєкту **openfootball**:
  `https://raw.githubusercontent.com/openfootball/worldcup.json/master/2026/worldcup.json`
- Сторінка опитує джерело **кожні 60 секунд** (з антикешем), сама перемальовує
  рахунок, переможців і добудовує сітку.
- Бейдж **LIVE** світиться лише у вікні матчу (за годинником, незалежно від даних).
- Час показано в **центральноєвропейському** поясі (CET/CEST, авто-перехід на літній час),
  pub-friendly матчі 17:00–23:59 позначені 🍺.

> Примітка: openfootball — це дані, які ведуть волонтери через git-коміти, а не
> миттєвий live-фід. Рахунок з'являється з невеликою затримкою (плюс ~5 хв CDN-кешу GitHub).

## Локальний запуск

Через подвійний клік `fetch` може блокуватися — краще підняти локальний сервер:

```bash
python3 -m http.server 8000
# відкрити http://localhost:8000
```

## Деплой

Чиста статика — підходить для GitHub Pages, Cloudflare Pages, Netlify тощо.
`index.html` має лежати в корені. Жодних змінних оточення чи build-команд не потрібно.
